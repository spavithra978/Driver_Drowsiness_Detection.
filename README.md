# 🚗 Driver Drowsiness Detection

## 📌 Project Overview
This project detects driver fatigue using deep learning by analyzing eye closure and yawning patterns.

## 🧠 Models Used
- Custom CNN
- MobileNetV2 (Transfer Learning)

## 📊 Results
- CNN Accuracy: 80.5%
- MobileNetV2 Accuracy: 87.0%

## 🔍 Features
- Eye state detection (Open / Closed)
- Yawning detection
- Fatigue classification (Alert / Mild / Severe)
- Fatigue progression graph

## 🛠️ Technologies
- Python
- TensorFlow / Keras
- OpenCV
- Matplotlib

## 📈 Output
The system predicts driver fatigue levels and visualizes progression over time.

## 🚀 Future Scope
- Real-time detection using webcam
- Integration with vehicle safety systems
